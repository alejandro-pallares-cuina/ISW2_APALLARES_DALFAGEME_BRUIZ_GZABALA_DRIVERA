# ISW2_APALLARES_DALFAGEME_BRUIZ_GZABALA_DRIVERA

Partiendo del proyecto Relecloud desarrollado individualmente en la primera parte del curso, deberéis crear por equipos, de un máximo de cinco integrantes, una versión extendida alojada en un repositorio de Azure DevOps.

El objetivo es aplicar prácticas profesionales de desarrollo colaborativo, integración continua y entrega continua (CI/CD) sobre una aplicación Django desplegable en Azure App Service.
